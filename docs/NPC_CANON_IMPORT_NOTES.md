# 超电磁炮篇 NPC 导入说明

本次根据客户提供的 `超电磁炮篇人物角色卡.rar`，结合《某魔法的禁书目录 / 某科学的超电磁炮》原著设定，将角色资料整理为插件默认 NPC 种子。

## 导入原则

- 不把客户角色卡原文整段塞入数据库，避免每轮结算时上下文过长。
- 每名 NPC 只保留文游运行需要的摘要：身份、阵营、常驻地点、性格倾向、能力边界和剧情用途。
- 原作主角、Level 5、统括理事会、木原一族、暗部核心成员默认不作为玩家可驱使对象。
- 高危人物以“传闻、痕迹、管理员投放事件、禁区背景”出现，防止破坏玩家行动和地图规则。
- 玩家仍默认创建原创角色，不能直接扮演原作人物本人。

## 高危出场限制

以下 NPC 默认不会参与普通随机移动，避免每小时结算时出现“亚雷斯塔逛商店街”“垣根帝督随机路过食堂”这类破坏世界观的情况：

- `disposition` 为“高危”“禁区”“传闻”“危险”的 NPC
- 统括理事会、木原一族相关 NPC
- 暗部相关 NPC
- 除御坂美琴、食蜂操祈外的 Level 5 NPC

这些角色仍可由管理员通过后台预设事件、公共事件或剧情安排投放。
