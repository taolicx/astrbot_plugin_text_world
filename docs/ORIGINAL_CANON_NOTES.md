本插件默认世界按《某魔法的禁书目录》系列中的“学园都市”进行同人文游化处理。

## 插件处理原则

- 默认舞台是科学侧的学园都市，玩家日常以第七学区为核心。
- 玩家角色默认是原创角色，不默认扮演原作人物本人。
- Level 5 属于极少数顶级超能力者，普通玩家创建时应优先限制在 Level 0 到 Level 4；Level 5 建议只允许管理员特批。
- Level 6、绝对能力者、魔神、统括理事长、原作主角本人、无代价幻想杀手等设定禁止作为普通角色卡通过。
- 原作高位人物可以作为偶遇、新闻、传闻或克制支援出现，但不能替玩家完成行动目标。
- 魔法侧可以作为访客、异常线索或长期伏笔出现，不默认压过科学侧日常主线。
- 插件程序给出的地图、位置、状态、货币、背包、相遇名单和事件效果高于世界书，AI 只负责包装叙事。

## 本轮已校准内容

- 默认地图地点改为原作第七学区设施和二十三学区结构。
- 默认 NPC 改为上条当麻、御坂美琴、初春饰利、白井黑子、佐天泪子、黄泉川爱穗、冥土追魂、茵蒂克丝等适合跑团互动的角色与传闻接触点。
- 默认事件增加 Seventh Mist、风纪委员第177支部、学舍之园、栅川中学、警备员封锁线等原作风格公共事件。
- 角色卡模板和自然语言整理提示词增加 Level 5/Level 6、原作人物本人、科学侧/魔法侧边界约束。
