# 群文游文字世界：所有指令详细说明书

适用插件：`astrbot_plugin_text_world`
适用版本：`0.4.0`
适用主题：学园都市 / 第七学区小时制群文游

这份说明书只写“怎么用指令”。玩家可以直接照着发，管理员可以照着开世界、审核角色、触发事件和结算。

## 1. 先看规则

世界开启后，群里普通聊天不会触发机器人回复。
也就是说，玩家在群里正常聊天、吐槽、闲聊，插件会尽量静默，不会抢正常对话。只有发送本文列出的文游指令，机器人才会回复。

玩家私聊机器人并发送 `绑定文游私聊` 后，私聊也会进入文游静默模式：只回复文游指令，普通私聊内容会被忽略，不再走正常聊天输出。

每个玩家每轮只能提交 1 次行动。默认 1 小时结算 1 轮。管理员可以手动结算。公共事件默认约 2 小时触发一次，也可以由管理员手动触发或在后台加入下一轮。

个人行动结果会优先私聊发送。群里只发送公共公告、大事件、传闻、聚集点等所有人能知道的信息。想稳定收到私聊结果，玩家需要先私聊机器人发送 `绑定文游私聊`。

`行动` 可以在群里提交；玩家私聊机器人发送 `绑定文游私聊` 成功后，也可以在私聊里提交到已绑定的群世界。私聊里发送 `创建角色`、`世界开启`、`世界结算`、`世界事件`、`世界后台` 会被明确拒绝。

角色创建后默认需要管理员审核。未审核通过前，可以看状态，但不能正常提交行动、签到也会受限。

## 3. 玩家最快上手流程

```text
1. 在群里发送：世界帮助
2. 在群里发送：角色卡模板
3. 在群里发送：创建角色 游戏名 | 身份 | 阵营 | 能力 | 能力等级 | 穿衣着装 | 简易身材
4. 等管理员审核通过
5. 私聊机器人发送：绑定文游私聊
6. 在群里发送：地图
7. 每轮在群里或绑定后的私聊发送：行动 你这一小时想做什么
8. 等待整点或管理员结算
9. 群里看公共公告，私聊看个人结果
10. 每天发送：签到
```

## 4. 管理员最快上手流程

```text
1. 在目标群发送：世界开启
2. 发送：世界后台
3. 打开后台并登录管理员账号
4. 审核玩家角色卡，必要时修改 QQ 绑定、身份、能力、位置、状态
5. 在后台添加事件、商品、NPC 或查看历史
6. 需要立刻推进剧情时发送：世界结算
7. 需要下一轮强制发生公共事件时发送：世界事件
```

## 6. `角色卡模板`

可用指令：

```text
角色卡模板
角色模板
创建格式
```

使用位置：群聊、私聊都可以。
作用：显示角色创建格式、字段解释和示例。

标准格式：

```text
创建角色 游戏名 | 身份 | 阵营 | 能力 | 能力等级 [| 穿衣着装 | 简易身材]
```

字段解释：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 游戏名 | 必填 | 玩家在文游里的名字，也会默认作为 Web 前端登录账号 |
| 身份 | 必填 | 角色在学园都市里的身份，例如学生、风纪委员见习、研究所实习生等 |
| 阵营 | 建议填 | 普通学生、风纪委员 Judgment、警备员 Anti-Skill、研究机构、Skill-Out 等 |
| 能力 | 建议填 | 能力名称、表现、限制、代价。通常一名能力者只有一种主能力 |
| 能力等级 | 建议填 | Level 0 到 Level 4 常规可用，Level 5 建议只允许管理员特批，Level 6 禁止 |
| 穿衣着装 | 可选 | 校服、外套、标志性穿搭，状态栏和剧情会使用 |
| 简易身材 | 可选 | 身高、体型、发型等低敏外观描述，状态栏会显示 |

## 7. `世界开启`

可用指令：

```text
世界开启

文游开启
```

使用位置：群聊。
推荐权限：管理员使用。
作用：在当前群登记并开启一个文字世界。

发送后会返回当前轮数和后台地址。世界开启后：

```text
群内普通聊天会被静默处理。
群内文游指令会正常回复。
每轮默认 1 小时自动结算。
公共事件默认约 2 小时检查一次。

```

示例：

```text
世界开启
```

常见提示：

| 提示 | 含义 |
| --- | --- |
| `请在群聊中使用。` | 这个指令不能在私聊开启世界 |
| 返回后台地址 | 世界已登记，可以进入后台管理 |

## 8. `创建角色`

可用指令：

```text
创建角色
角色创建
创建人物
```

使用位置：群聊。
权限：玩家。
作用：提交或更新自己的角色卡。

### 8.1 标准格式

```text
创建角色 游戏名 | 身份 | 阵营 | 能力 | 能力等级 | 穿衣着装 | 简易身材
```

完整示例：

```text
创建角色 星野遥 | 第七学区某高中学生，风纪委员见习 | 风纪委员 Judgment | 微弱电磁感应，可察觉近距离异常电流，连续使用会头痛 | Level 2 | 常规学生制服，深色外套 | 160cm，偏瘦，黑长发
```

最低可用格式：

```text
创建角色 星野遥 | 第七学区某高中学生
```

最低格式可以提交，但为了剧情和后台状态栏完整，建议尽量填满身份、阵营、能力、等级、着装、身材。

### 8.2 自然语言格式

插件支持自然语言创建角色，会尝试接入快速模型整理成标准角色卡。

```text
创建角色 我叫星野遥，是第七学区某高中学生兼风纪委员见习，属于风纪委员阵营，能力是微弱电磁感应，可以察觉近距离异常电流，连续使用会头痛，能力等级是 Level 2，平时穿学生制服和深色外套，身高160cm，黑长发。
```

整理成功后，机器人会回复类似：

```text
已按自然语言整理角色卡：
游戏名：星野遥
身份：第七学区某高中学生，风纪委员见习
阵营：风纪委员 Judgment
能力：微弱电磁感应，可察觉近距离异常电流，连续使用会头痛
能力等级：Level 2
穿衣着装：学生制服和深色外套
简易身材：160cm，黑长发
角色卡已提交，等待管理员审核。
```

### 8.3 审核规则

创建后默认是 `待审核`。管理员需要在 Web 后台把角色审核为 `已通过`，玩家才能正式行动和签到。

如果同一个 QQ 再次发送 `创建角色`，会更新原角色卡，并重新等待审核。
如果游戏名已被别人使用，会提示换一个名字。

### 8.4 角色卡限制

角色卡不能写明显越权设定，例如：

```text
我是 Level 6
我是绝对能力者
我是统括理事会成员
我能自由进入没有窗户的大楼
我能控制所有 NPC
我开局拥有无限学都币
```

常见提示：

| 提示 | 含义 |
| --- | --- |
| `请在群里提交角色卡。` | 创建角色不能在私聊使用 |
| `格式：创建角色 游戏名 | 身份 | 阵营 | 能力 | 能力等级` | 必填字段不足 |
| `这个游戏名已经被使用了，请换一个。` | 角色名被别人占用 |
| `角色卡已提交，等待管理员审核。` | 提交成功，需要管理员审核 |
| `角色卡已更新，等待管理员审核。` | 你更新了原角色卡，需要重新审核 |

## 9. `行动`

可用指令：

```text
行动 你的本轮行动
文游行动 你的本轮行动
提交行动 你的本轮行动
```

使用位置：群聊。
权限：已审核玩家。
作用：提交本轮行动，每轮只能提交 1 次。

默认每轮 1 小时，所有玩家的行动会在结算时统一执行。去同一个地点的玩家可能相遇，玩家和 NPC 到同一地点也可能触发剧情。

### 9.1 基本写法

```text
行动 我前往第七学区公共广场，观察新闻屏幕附近有没有异常聚集。
```

```text
行动 我留在学生寮休息，补充水分和饱食度，顺便整理背包。
```

```text
行动 我去 Seventh Mist 购买饮用水和便当。
```

```text
行动 我去冥土追魂所在医院治疗伤势。
```

### 9.2 推荐行动类型

| 类型 | 示例 | 可能效果 |
| --- | --- | --- |
| 移动 | `行动 我从校门去主干道，再观察附近情况。` | 按地图路线移动，路线不成立会被拦截 |
| 调查 | `行动 我去资料室查询最近的能力开发异常记录。` | 获取线索、传闻或公开资料 |
| 互动 | `行动 我在广场等星野遥，想和她交换情报。` | 与玩家或 NPC 互动，可能影响好感度 |
| 购买 | `行动 我去商店买饮用水和便当。` | 消耗学都币，获得物品或恢复状态 |
| 治疗 | `行动 我去医院处理伤口。` | 消耗学都币，恢复生命 |
| 打工 | `行动 我在商店街做一小时兼职。` | 获得学都币，消耗精力 |
| 休息 | `行动 我回宿舍休息，早点睡觉。` | 恢复精力和心情 |
| 能力开发 | `行动 我参加能力开发课程，练习个人现实演算。` | 增加能力经验，每天正式训练有限制 |
| 可控冲突 | `行动 我在训练场进行低风险对练。` | 增加战斗经验，会有接触损伤 |
| 玩家冲突 | `行动 我攻击/迎战某玩家。` | 会按 PvP 处理，生命和精力都会明显下降 |
| 风险行动 | `行动 我尝试潜入研究所封锁区外围调查。` | 高风险，可能受伤或被限制，只会获得外围线索 |

### 9.3 行动写作建议

写清楚三件事最容易结算：

```text
我要去哪里？
我要做什么？
我想和谁互动或调查什么？
```

推荐格式：

```text
行动 我从当前位置前往【地点】，想做【事情】，如果遇到【玩家/NPC/线索】就尝试【互动方式】。
```

示例：

```text
行动 我从校门前往风纪委员第177支部，报告昨天在商店街听到的可疑传闻，并询问是否需要协助巡逻。
```

### 9.4 行动限制

每轮只能提交一次。已提交后不能用指令覆盖，除非管理员从后台处理数据库或等下一轮。

行动不能太长，默认最多约 120 字。太长会提示缩短。

地图有路线约束，不能突然从第七学区校门瞬移到没有连接的禁区。路线不成立时，本轮会留在原地。

生命过低时，只能休息或去医院治疗。

高风险行动、互殴、袭击和正面冲突会导致生命、精力、心情明显下降。涉及暗部、统括理事会、没有窗户的大楼、核心研究机密等内容时，通常只能得到外围痕迹，不会一轮直接揭开核心真相。

如果你的行动影响到其他玩家，例如殴打、袭击、强行拦截、一起行动等，目标玩家也可能收到私聊反馈。即使对方本轮没有提交行动，只要状态受影响，也会尽量私聊通知对方。

玩家交互会区分同行、交流、支援保护、压力型互动、赠与/交易和冲突。写清楚对象、方式和物品/金额，系统会给双方私聊反馈，并小幅影响关系分。

交易和赠与只认“支出方确认”：你只能写自己给别人、自己转账、自己赠送背包物品。不能单方面写“对方给我 1000 学都币”。抢夺、偷取、勒索可以作为冲突行动尝试，但不会直接把对方的钱或物品转给你。

### 9.5 常见提示

| 提示 | 含义 |
| --- | --- |
| `行动需要在群里提交；如果想私聊提交，请先私聊发送：绑定文游私聊。` | 私聊还没有绑定到群世界 |
| `检测到你在多个群世界都有角色` | 私聊无法判断要向哪个群世界提交行动 |
| `行动不能为空。` | `行动` 后面没有写内容 |
| `行动太长了` | 超过配置长度，需要缩短 |
| `你还没有角色卡。` | 需要先创建角色 |
| `你的角色卡还没有通过审核，暂时不能行动。` | 等管理员审核 |
| `你本轮已经提交过行动了。每小时只能提交一次。` | 本轮已用掉行动 |
| `地图规则拦截` | 目标地点不能从当前位置直接到达 |
| `生命值过低，本轮只能休息或去医院治疗。` | 角色状态危险，需要恢复 |

## 11. `地图`

可用指令：

```text
地图
世界地图
```

使用位置：群聊；绑定私聊后也可以私聊使用。
权限：玩家。
作用：查看地图图片和文字地图。

当前版本会优先发送图片版地图：

```text
学园都市总览图
第七学区详细图
路线连接图
```

如果地图图片缺失，会回退为文字地图，显示：

```text
当前位置
当前位置描述
可前往地点
全部地点连接
```

示例：

```text
地图
```

地图使用原则：

```text
行动移动时，请尽量写地图里存在的地点。
想去相邻地点，成功率最高。
想跨区移动，建议经过交通节点或说明路线。
禁区、研究所、学舍之园、没有窗户的大楼等地点有权限边界。
```

常见提示：

| 提示 | 含义 |
| --- | --- |
| `没有找到已绑定的群世界。` | 私聊查地图前还没有绑定 |
| `检测到你在多个群世界都有角色` | 私聊不知道查哪个世界 |
| 群里发出地图图片 | 图片地图正常 |
| 返回文字地点连接 | 地图图片缺失或无法发送时的兜底 |

## 12. `待结算`

可用指令：

```text
待结算
本轮行动
行动列表
```

使用位置：群聊；绑定私聊后也可以私聊使用。
权限：玩家。
作用：查看当前轮次、本轮已提交行动人数、已审核角色数、下次结算时间。

示例：

```text
待结算
```

返回内容类似：

```text
【待结算】
当前轮次：第 3 轮
已提交行动：5 人
已审核角色：8 人

```

说明：

```text
这个指令不会显示每个人具体提交了什么。
它只显示数量，避免泄露玩家私密行动。
```

## 13. `签到`

可用指令：

```text
签到
每日签到
```

使用位置：群聊；绑定私聊后也可以私聊使用。
权限：已审核玩家。
作用：每天领取学都币，并小幅提升心情。

默认奖励：`20` 学都币。实际数值以插件配置 `checkin_reward` 为准。

示例：

```text
签到
```

```text

```

## 14. `绑定文游私聊`

可用指令：

```text
绑定文游私聊
绑定世界私聊
文游私聊绑定
```

使用位置：私聊机器人。
权限：玩家。
作用：把当前 QQ 的文游角色绑定到这个私聊会话，用于接收个人结算结果、早上 8 点状态栏，以及在私聊里使用 `状态`、`地图`、`待结算`、`签到`。

推荐流程：

```text
1. 先在群里创建角色
2. 等管理员审核通过
3. 私聊机器人：绑定文游私聊
4. 之后个人结果会尽量发到私聊
```

成功返回：

```text
已绑定文游私聊，关联角色数：1。
```

常见提示：

| 提示 | 含义 |
| --- | --- |
| `还没有找到你的角色卡` | 还没在群里创建角色，或 QQ 没有关联角色 |
| `检测到你在多个群世界里都有通过审核的角色` | 一个 QQ 在多个群都有角色，私聊无法自动判断绑定哪一个 |

如果同一个 QQ 在多个群都有角色，建议：

```text
让管理员确认需要绑定的群世界。
确保当前私聊结果对应唯一角色。
必要时在后台调整角色绑定。
```

## 19. 群聊静默规则

世界开启后，如果配置 `enable_world_chat_silence` 为开启，群聊会进入文游静默模式。玩家私聊发送 `绑定文游私聊` 成功后，私聊也会进入文游静默模式。

会回复：

```text
世界帮助
角色卡模板
创建角色 ...
行动 ...
状态
地图
待结算
签到
管理员指令
```

不会回复：

```text

吐槽
不带指令的剧情讨论
玩家之间闲聊
```

这样做是为了防止普通聊天被大模型抢答，避免群里刷屏。

## 26. 常见问题

### 26.1 群里说话机器人没反应

这是正常现象。世界开启后，普通聊天会静默。请使用文游指令，例如：

```text
行动 我去第七学区公共广场调查。
状态
地图
```

### 26.2 群里创建了角色，私聊签到却说没角色

先私聊机器人：

```text
绑定文游私聊
```

如果一个 QQ 在多个群世界都有角色，私聊可能无法判断要绑定哪个，需要管理员在后台确认或调整。

### 26.3 行动提交不了

按顺序检查：

```text
是否在群里发送？
是否已经创建角色？
角色是否已审核通过？
本轮是否已经提交过行动？
行动内容是否太长？
生命值是否过低？
是否触发反作弊或越权描述？
```

### 26.4 地图路线不成立

先发送：

```text
地图
```

查看当前位置和可前往地点。下一轮按相邻地点移动，或者写清楚交通路线。

### 26.5 收不到个人结果

先私聊机器人：

```text
绑定文游私聊
```

如果仍收不到，可能是平台私聊权限、OneBot 私聊发送能力、机器人好友关系或插件配置问题。可以让管理员查看后台历史，个人结果通常会写入历史记录。

### 26.6 可以改行动吗

当前聊天指令没有“修改本轮行动”。每轮提交后就锁定，下一轮再提交新行动。确实需要改时，让管理员后台处理。

### 26.7 可以删除角色吗

当前聊天指令没有“删除角色”。管理员可以在后台调整审核状态、改名、改绑定或重置数据。

### 26.8 可以交易吗

当前聊天指令没有独立交易指令。可以用 `行动` 描述交易意图，由结算处理；更完整交易系统属于后续扩展。
